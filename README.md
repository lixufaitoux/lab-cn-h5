# lab-cn-h5

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不依赖于特定域名或网站，仅作为静态资源存放与展示。

## 项目简介

本仓库收集并维护一系列独立的 H5 页面，涵盖不同场景下的前端实现。每个页面均为独立的 HTML 文件，可直接在浏览器中打开运行，无需额外服务端支持。

## 目录说明

```
/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（图片、样式、脚本等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面。
- `assets/`：页面共享的静态资源，按类型或用途组织。
- `docs/`：关于页面设计、使用方式或更新记录的附加文档。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，尽可能减少外部依赖。
- 页面可能引用 `assets/` 目录下的公共资源，或使用 CDN 加载库文件。
- 页面内容以归档和展示为目的，不包含任何商业推广或敏感信息。
- 部分页面为早期版本，可能存在兼容性或样式差异。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下对应的 HTML 文件。
3. 如需查看在线版本，可通过 GitHub Pages 或其他静态托管服务部署。
